因项目默认使用ollama qwen2.5:1.5b模型，所以需要提前拉取模型​

# 基于 docker-compose.yaml 配置文件，ollama拉取
ollama pull deepseek-r1:1.5b
ollama pull qwen2.5:1.5b
# 基于 docker-compose-ollama.yaml 配置文件，使用docker拉取
docker exec ollama pull qwen2.5:1.5b
docker exec ollama pull deepseek-r1:1.5b

如果不需要知识库AI问答功能，可以修改 docker-compose.yaml 或 docker-compose-ollama.yaml 关闭ollama对话和嵌入功能，以节省资源​

# 关闭ollama对话
SPRING_AI_OLLAMA_CHAT_ENABLED: false
# 关闭ollama嵌入
SPRING_AI_OLLAMA_EMBEDDING_ENABLED: false

修改配置，否则上传图片、文件和知识库无法正常显示​

  • 修改上述编排文件
# 请将服务器127.0.0.1替换为你的服务器ip，或者域名
BYTEDESK_UPLOAD_URL: http://127.0.0.1:9003
BYTEDESK_KBASE_API_URL: http://127.0.0.1:9003
