Android 集成指南

部分功能​

  • 微语官方技术支持
  • 100%全部开源，支持自定义界面
  • 支持人工客服
  • 支持机器人
  • 支持文字、图片、语音、表情
  • 支持消息预知：对方正在输入
  • 支持消息状态：送达、已读
  • 支持消息撤回
  • 对接第三方账号系统/多用户切换
