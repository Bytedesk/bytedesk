## 介绍

### 企业IM

- 局域网即时通讯
- 企业成员管理
- 聊天记录监控
- 自定义通知，如：接收运维报警、股票实时报价等
- 更利于企业内部员工之间协作

### 知识库AI对话

- 可编排AI知识库问答
- 对接知识库
- 大模型对话
- 拖拽式配置
- 可对接调用第三方知识库问答系统接口
- 实时调用自有业务系统Api
- 实现系统解耦，更利于扩展

### 智能客服 + 人工客服

- 智能客服
- 实现人工客服兜底
- 提供转人工客服接口，可为第三方AI知识库问答系统提供转人工解决方案
- 支持多渠道
