# 培训模块

智能培训体系：搭建智能陪练模块，整合业务知识学习资源，提供线上考试功能；新入职员工完成学习考试后，可通过AI陪练模拟真实业务场景，逐步实现独立上岗。

- 客服培训
- 员工培训

## 培训管理

- [] 发布业务培训课程，支持线上学习（PC 端 / 移动端）；
- [] 组织实操模拟演练，模拟咨询接待、异议处理、智能工具使用场景；
- [] 记录培训参与情况，生成考勤报表；
- [] 结合智能质检结果，针对客服薄弱环节（如业务准确度低）定制培训内容；
- [] 微视频培训及学习记录统计等
- [] 培训内容需覆盖最新业务规则、设备知识及智能系统操作；
- [] 培训课程需搭配考核题库（每课程至少 30 题），确保学习效果；
- [] 培训参与率需≥95%，未参与客服需补学

## 业务能力考核

- [] 发起线上考核，题目涵盖业务知识、系统操作（智能工具使用、工单处理）；
- [] 自动评分并生成考核报告，标注薄弱环节（如智能派单操作不熟练）
- [] 考核不通过的客服限制上岗，需重新培训
- [] 关联客服绩效及智能质检结果，形成能力评估矩阵
- [] 与客服主管共享考核结果，关联绩效评定
- [] 考核次数每月至少 1 次，新客服上岗前需额外考核 1 次
- [] 考核报告需包含提升建议，指导客服针对性学习
