# 群组

## 主要功能包括

### 基本群组管理

- 创建/解散群组
- 更新群组信息
- 管理群组设置

### 成员管理

- 邀请新成员
- 移除成员
- 添加/移除管理员
- 成员身份检查

### 群组权限

- 邀请权限控制
- 管理员权限
- 全员禁言管理

### 邀请机制

- 发送邀请
- 处理邀请
- 邀请状态跟踪
- 邀请过期机制
