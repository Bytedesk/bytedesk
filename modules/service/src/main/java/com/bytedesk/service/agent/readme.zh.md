# 客服

## 主要功能包括

- 状态管理：
  - 在线/离线状态切换
  - 忙碌状态设置
  - 自动状态检查
  - 最后活跃时间记录
- 会话管理：
  - 最大会话数限制
  - 当前会话数统计
  - 会话分配检查
  - 会话开始/结束处理
- 统计功能：
  - 服务访客数统计
  - 平均响应时间计算
  - 满意度评分记录
  - 工作量统计
- 工作组管理：
  - 添加/移除工作组
  - 技能检查
  - 专业分组
- 自动化：
  - 定时状态检查
  - 超时自动离线
  - 忙碌状态自动解除

## 这个设计

- 提供了完整的客服状态管理
- 支持智能会话分配
- 包含详细的统计功能
- 实现了工作组管理
- 具有自动化特性
