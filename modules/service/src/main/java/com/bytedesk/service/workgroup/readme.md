# 工作组

## WorkgroupEntity和skills的区别主要在于

1. 组织结构 vs 能力标签

- WorkgroupEntity是组织结构层面的分组,比如"售前组"、"售后组"、"技术支持组"等
- Skills是能力标签层面的标识,比如"Java"、"Python"、"数据库"等技术能力
2.分配策略不同

## 使用场景

### Workgroup用于

- 业务部门划分
- 工作时间安排
- 团队管理
- 负载均衡
- 权限控制

### Skills用于

- 专业能力匹配
- 问题精准分配
- 服务质量保证
- 客服培训发展
- 多语言支持

## 主要区别总结

- Workgroup是"组织"维度
- Skills是"能力"维度
- Workgroup控制"如何分配"
- Skills控制"分配给谁"
- Workgroup关注团队管理
- Skills关注服务质量
