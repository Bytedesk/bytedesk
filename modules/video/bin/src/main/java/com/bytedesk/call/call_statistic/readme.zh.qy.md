呼入的话务总量

呼出量
呼出的话务总量

## 呼入情况

当前排队量
当前正在排队的话务总量

平均排队时长
产生排队的话务中，排队时长总和与排队次数的比值

正在通话量
当前正在通话的呼入话务总量

呼入接通量
呼入并接通的话务总量

服务时间呼入接通率
服务时间内，呼入接通量与呼入量的比值

呼入时长
呼入通话的总时长

队列接通率
进入队列并接通的话务量与进入队列话务量（含客户快速挂机量）的比值

X秒振铃摘机率
客服接起前振铃时长小于等于振铃摘机统计周期的话务量与振铃量（不含客户快速挂机量、临界挂机数）的比值

X秒客服振铃摘机率
客服接起前振铃时长（去除用户排队时长）小于等于振铃摘机统计周期的话务量与振铃量（不含客户快速挂机量、临界挂机数）的比值。

一次性解决率
统计周期内，访客未二次来电或来电的咨询分类不同的话务量与呼入量的比值，计算规则可在“设置-统计规则”处配置

X秒队列接通率
设定接通率统计周期内，进入队列并接通的话务量与进入队列话务量（含客户快速挂机量）的比值，其中“x秒”的统计包含排队时间与振铃时间

重复进线（队列）率
统计周期内，访客重复呼入并进入人工队列的话务量与呼入量的比值

平均呼入排队时长
所有呼入的话务中，排队时长总和与所有呼入量的比值

平均摘机时长
摘机时长（从振铃到摘机的时间）总和与接听量的比值

平均呼入通话时长
呼入通话时长的平均值

平均呼入处理时长
呼入的通话，“处理中”状态时长总和与呼入“处理中”状态总次数的比值

转人工呼入量
访客呼入且系统分配应当由人工接待的话务总量（相比”队列呼入量“指标，转人工呼入量剔除掉了仅IVR相关的数据，在关注人工服务的视角下更适用）

转人工呼入接通率
接通量与转人工呼入量的比值

客户快速挂机量

临界挂机数

IVR信息校验失败量

## 话费明细

1. 该报表仅统计纯云号码的话费明细

2. 呼入通话不计费，呼出通话计费，不足一分钟的扣除一分钟

3. 跨0点的通话，算作前一天的通话记录，计费时长应全部统计在前一天

4. 若整个通话过程中未发生转接，计费时长=当前坐席通话时长

5. 若整个通话过程中发生转接，前一通通话的计费时长=前一个坐席通话时长+后一个坐席的振铃时长；后一通通话的计费时长=后一个坐席通话时长

6. 若发生多方通话，计费时长=整个通话从建立到多方通话结束的时长+每个第三方的通话时长，被邀请的内部客服不做计费，被邀请的第三方号码需要做计费。多方通话的通话ID及类型显示为 --

7. 如果发生月租计费，相关扣费会记录在月租明细报表中展示，不会在消费报表中展示

8. 双向回呼模式，由于X呼叫A，X呼叫B分别会产生话费，因此每通双向回呼会话会产生两条计费记录

9. AXB模式，由于A呼叫X，X呼叫B分别会产生话费，因此按2倍通话时间计费
