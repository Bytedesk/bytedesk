# 介绍

:::tip
微语仍处于早期的快速迭代阶段，文档可能落后于开发，导致功能描述可能不符，以最新发布的软件版本为准
:::

## 微语是什么？

- 一款开源聊天软件
- 支持企业IM、在线客服、知识库、工单系统、AI Agent等
- 致力于实现企业管理软件一站式解决方案

## 具有哪些功能？

- 多层组织架构
- 角色管理
- 权限管理
- 聊天记录管理
- 同事对话
- 群组管理
