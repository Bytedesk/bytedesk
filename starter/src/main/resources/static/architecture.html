Technology Stack

技术栈

  • Java Version: Java版本: 17
  • Framework: 框架: Spring Boot 3.4.4
  • Web Server: Web服务器: Jetty (replacing Tomcat) Jetty (替代Tomcat)
  • Database Support: 数据库支持: MySQL, PostgreSQL, H2, Oracle
  • Caching: 缓存: Redis, Caffeine
  • Messaging: 消息系统: Apache Artemis, Spring Integration
  • Security: 安全: Spring Security, OAuth2, LDAP
  • API Documentation: API文档: SpringDoc OpenAPI
  • State Management: 状态管理: Spring Statemachine, COLA Statemachine
  • UI Technologies: UI技术: Freemarker, Thymeleaf
  • Monitoring: 监控: Spring Actuator, Prometheus
  • Container Support: 容器支持: Docker
