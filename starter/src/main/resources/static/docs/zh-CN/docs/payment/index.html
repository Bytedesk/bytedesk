价格说明​

  • 社区版：开源，免费，可商用，单租户，IM用户数量不限、客服数量限制50个、机器人限制50个，知识库数量50个
  • 企业版：包含开源社区版所有功能 + 高级功能，单租户，支持logo/品牌等自定义，功能不限，用户数量不限、客服数量不限、机器人数量不限，知识库数量不限，部署包（不含源码，源码需另行购买），一次性收费：3w
  • 平台版：包括企业版所有功能，支持多租户，支持logo/品牌等自定义，功能不限，用户数量不限、客服数量不限、机器人数量不限、知识库数量不限，部署包（不含源码，源码需另行购买），一次性收费：6w
  • 产品代理：购买企业版>=2份授权，支持logo/品牌等自定义，价格8折
  • 如有定制需求，请联系微信，发送您的需求，评估开发周期/定价
  • 以上价格均为一次性收费，1年之内免费升级，一年之后如需升级按照15%/年收取，自愿可选。
  • 源码价格：管理后台5w，客服工作台5w，手机客服端(含安卓/iOS)5w，h5访客端2w，服务器高级功能源码5w，可根据需要单独购买。如购买源码，则无需购买企业版或平台版。
  • 如果您只需要部分功能，价格可谈，请联系微信
  • 上述价格均不含税点，如需发票，需额外加收3%税点
